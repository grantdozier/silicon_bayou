# Source artwork — not deployed

Original client logos as supplied, with their backgrounds baked in (four over
black, three over white). `images/partners/*` are the un-composited,
transparent re-cuts generated from these; those are what the site actually uses.

Kept in version control because these are the only copies. Pruned from the
public deploy by `.github/workflows/deploy.yml` so ~3 MB of unused raster does
not ship.
